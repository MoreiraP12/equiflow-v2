Welcome to equiflow's documentation!
====================================

**equiflow** is a Python package for creating equity-focused cohort selection 
flow diagrams in clinical and ML research.

.. note::

   This project is under active development.

Check out the :doc:`usage` section for further information, including
how to :ref:`install <installation>` the project.
